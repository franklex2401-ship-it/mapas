Sitios Arqueológicos de Costa Rica no restringidos al publico

Descripción: Mapa web interactivo que localiza y describe seis sitios arqueológicos distribuidos en el territorio costarricense. Al hacer clic en cada punto, el usuario accede a breve información del sitio: nombre, descripción, tipo, período histórico, elementos patrimoniales y una fotografía representativa.

Justificación: Para la creación de este mapa se tomaron varias decisiones metodológicas y comunicativas orientadas a señalar la ubicación y las caracteristicas del patrimonio arqueológico costarricense.
En cuanto a los datos, se estructuró una base de datos en formato CSV con atributos que permiten caracterizar cada sitio desde múltiples dimensiones: su función (ceremonial, habitacional, rupestre), su período de ocupación, los elementos materiales presentes y su estado de conservación. Esta diversidad de atributos refleja la complejidad del registro arqueológico nacional.
La decisión de utilizar QGIS con exportación a qgis2web responde a la necesidad de crear un producto accesible sin requerir software especializado por parte del usuario. El mapa web permite una exploración muy breve del territorio, donde cada punto representa no solo una coordenada, sino un lugar que no es totalmente restringido a las visitas. 
La incorporación de imágenes fotográficas en cada popup fue una decisión deliberada: las fotografías muestran los datos y generan una conexión visual inmediata con el sitio, algo que los textos descriptivos por sí solos no logran. Ver el Monumento Nacional Guayabo o las esferas de Finca 6 comunica en segundos lo que párrafos enteros difícilmente transmiten.
Se espera que el usuario, al interactuar con el mapa, comprenda que Costa Rica posee un patrimonio arqueológico diverso y geográficamente distribuido, que trasciende el conocimiento popular, y que cada sitio representa una forma distinta de organización social y cultural de los pueblos antiguos que habitaron este territorio.
